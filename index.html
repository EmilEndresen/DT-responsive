Absolute units

Absolute are units that are fixed, examples of absolute units are: px, cm, mm, in, pc, pt.

EM and REM

The EM unit is a relative unit assigned depending on the font size of the parent element.
So if the font size of the element is 20px, 1em would be 20px, and 2em would be 40px

The REM unit is similar to the EM unit, but in difference from EM it is relative to the font size of the root element

The CH unit

The ch unit is equal to the width of the curront font's 0, so 17ch would be equal to 17 zeros
